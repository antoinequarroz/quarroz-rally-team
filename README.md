# Quarroz Rally Team

Site vitrine Nuxt 4 pour le Quarroz Rally Team.

## Stack

- Nuxt 4, Vue 3, TypeScript strict
- Tailwind CSS
- Pinia + persisted state
- Nuxt i18n
- Color mode
- PWA via `@vite-pwa/nuxt`

## Commandes

```bash
npm install
npm run dev
npm run lint
npm run build
npm run preview
```
